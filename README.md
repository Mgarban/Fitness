Fitness Tracker
A full-stack Fitness Tracker application to log workouts, track exercise progress, and visualize statistics. 
Supports user authentication with email/password and Google OAuth, workout creation, history, and a dashboard with stats.

Features
-User Authentication
-Signup/Login with email and password
-Google OAuth login
-JWT-based session management

Workout Management
-Create workouts with type, duration, calories
-Log exercises: sets, reps, weight, distance
-Edit/Delete workouts

Workout History
-View workouts by day/week/month

Statistics Dashboard
-Total workouts
-Total calories burned
-Total volume lifted (sets × reps × weight)

Responsive UI
-Mobile-friendly design with TailwindCSS
-Simple and intuitive interface

Tech Stack
